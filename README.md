# TypMeter

A modern, fast, and minimalistic typing speed test application built with **Next.js**.

## Features

- **Typing Tests**: Test your speed in **Time** (15s, 30s, 60s) or **Words** (10, 25, 50, 100) modes.
- **Theme Support**: Beautiful **Light** and **Dark** modes with seamless switching.
- **Leaderboard**: Compete with others and see daily top 3 scores.
- **History**: Track your progress over time with detailed stats (WPM, Accuracy, Consistency).
- **Authentication**: Sign in securely to save your scores automatically.
- **Sound Effects**: Satisfying mechanical keyboard sounds (toggleable).

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Authentication**: Clerk
- **Icons**: Iconify

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Visit [http://localhost:3000](http://localhost:3000) to start typing!
